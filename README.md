🤖 Hand Gesture Controlled Computer System

An AI-powered computer vision application that enables users to control a computer using hand gestures instead of a traditional mouse and keyboard. The system leverages OpenCV, MediaPipe, and Python to recognize hand gestures in real time and perform various computer operations.

Note: This project was developed collaboratively as a team project. This repository is maintained as my individual portfolio copy with permission from my teammate.

🚀 Features
🖱️ Cursor Movement
👆 Left Click
👉 Right Click

📸 Screenshot Capture

Two Fingers	Adjust Brightness
Three Finger Swipe	Scroll
